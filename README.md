### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

- `src/components/` - React components
- `src/data/` - Local question data (will be replaced with Supabase)
- `src/types.ts` - TypeScript type definitions

## Game Flow

1. **Start Page**: Enter name and select language
2. **Countdown**: 3-2-1 countdown before quiz starts
3. **Quiz**: Answer questions one by one with timer
4. **Results**: View score, time, and return home
